# Blur안드로이드 앱
이 앱은 스마트 미러에 정보를 입력하기위해 제작된 앱 이다.

## 설치 방법

이 프로젝트를 빌드하고 실행하려면 다음 단계를 따르세요:
1. 이 저장소를 클론합니다.
2. Android Studio에서 프로젝트를 엽니다.
3. 적절한 SDK 및 도구를 사용하여 프로젝트를 빌드합니다.
4. 애플리케이션을 에뮬레이터나 연결된 장치에서 실행합니다.
