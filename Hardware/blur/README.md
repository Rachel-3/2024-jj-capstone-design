# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts

## `yarn dev`
Blur앱 개발자 모드에서 실행.
localhost:3000 통해 실행, 실행에 시간 걸림.

## `yarn build && electron . `
Blur앱 빌드 및 일렉트론 실행
빌드는 코드 수정 됐을 때만 사용

## `electron .`
빌드된 Blur앱 실행.
바로 실행.
빌드 완료되면 이 명령어만 실행하면 됨.
